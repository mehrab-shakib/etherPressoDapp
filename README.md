
# EtherPresso

A simple dApp (Decentralized Application) which is integrated with "MetaMask Wallet" built using solidity, hardhat, ether.js and react.js. 




## Features

- MetaMask Integration 
- Live Transaction 
- Sending Messages
- Transaction History
